The **Employee Payroll Management System** is a **console-based Java application** developed to understand and implement **Object-Oriented Programming (OOP) concepts** in a real-world scenario.  
This system allows managing employee details and calculating payroll based on salary structure, deductions, and allowances.

The project is designed as a **learning project**, focusing on clean code structure, OOP design, and logical implementation rather than UI complexity.

## üìö Key OOP Concepts Used

### 1. Encapsulation

-   Employee data is kept private
    
-   Accessed using getters and setters
    
-   Improves data security and maintainability
    

### 2. Inheritance

-   Common employee behavior can be extended for different employee types
    
-   Reduces code duplication
    

### 3. Polymorphism

-   Same method name used for different payroll calculations
    
-   Improves flexibility and scalability
    

### 4. Abstraction

-   Payroll logic is separated from user interaction
    
-   Makes the code easier to understand and modify
